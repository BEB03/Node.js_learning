## HTTP 서버 구축

http는 노드의 **내장 모듈**이다. 즉 따로 설치할 필요 없음. (1장의 웹 서버가 내장되어 있다는 게 이 말이었구나)
노드 실행 후 http라고 쳐서 정보를 확인해 볼 수 있었음.

```js
const http = require("http");
```

따라서 이렇게 시작하게 된다.
